# 棒読みタイマー

## このアプリについて

棒読みタイマーは、設定した時刻になると棒読みちゃんが音声でお知らせしてくれる便利なアプリです。
作業の区切りや、配信の時間管理などに便利です。

## はじめに（初めての方向け）

### 必要なもの

- **棒読みちゃん**
  - まだお持ちでない方は[棒読みちゃんの公式サイト](https://chi.usamimi.info/Program/Application/BouyomiChan/)からダウンロードしてください
  - 棒読みちゃんはタイマー通知を音声で読み上げるために必要です

- **Python**
  - このアプリの実行にはPythonが必要です
  - まだインストールしていない方は[Pythonの公式サイト](https://www.python.org/downloads/)からダウンロードしてください
  - インストール時に「Add Python to PATH」にチェックを入れることをお勧めします

### タイマーの起動方法

#### 方法1: コマンドプロンプトから起動（基本の方法）

1. まず棒読みちゃんを起動しておきます
2. コマンドプロンプトを開きます（Windowsキー+Rを押して「cmd」と入力）
3. 棒読みタイマーのフォルダに移動します（cdコマンドを使用）
   ```
   cd 棒読みタイマーがあるフォルダのパス
   ```
4. 次のコマンドでタイマーを起動します
   ```
   python timer.py config/sample_config.json
   ```

#### 方法2: 簡単に起動するための準備（一度だけ設定）

毎回コマンドを入力するのが面倒な場合は、以下の手順で簡単に起動できるバッチファイルを作成できます。

1. メモ帳などのテキストエディタを開きます
2. 以下の内容をコピー＆ペーストします
   ```
   @echo off
   echo 棒読みタイマーを起動します...
   python timer.py config/sample_config.json
   pause
   ```
3. ファイルを「起動.bat」という名前で棒読みタイマーのフォルダと同じ場所に保存します
4. これで「起動.bat」をダブルクリックするだけでタイマーを起動できます

### タイマーの使い方（4ステップ）

1. **棒読みちゃんを起動する**
   - 起動したら最小化しておいてもOK

2. **タイマーアプリを起動する**
   - 上記の方法でタイマーを起動します

3. **タイマーを設定する**
   - 画面下の「設定」ボタンをクリック
   - 好きなプリセットを選ぶか、時刻を設定
   - 「タイマー追加」ボタンをクリック
   - 「保存」ボタンをクリック

4. **待つだけ！**
   - 設定した時刻になると、自動で音声が流れます

## 詳しい使い方

### プリセットを使う（かんたん設定）

1. 「設定」ボタンをクリック
2. 「プリセット」から選びます:
   - **1時間ごと** - 次の正時（○時00分）にタイマー設定
   - **30分ごと** - 次の30分区切り（○時00分または○時30分）に設定
   - **15分ごと** - 次の15分区切り（○時00分、15分、30分、45分）に設定
3. 「タイマー追加」ボタンをクリック
4. 「保存」ボタンをクリック

### 時刻を自分で設定する

1. 「設定」ボタンをクリック
2. 「時刻指定」で何時何分かを設定:
   - 「時」と「分」を↑↓ボタンで選択
   - （分は5分単位で設定できます）
3. 「通知メッセージ」に読み上げてほしい言葉を入力
4. 「タイマー追加」ボタンをクリック
5. 「保存」ボタンをクリック

### タイマーの管理

- **タイマーの確認**: 設定画面の「登録済みタイマー」に表示されます
- **タイマーの削除**: 削除したいタイマーをクリックして「選択を削除」ボタン
- **全部消す**: 「全クリア」ボタン

## こんな使い方ができます

### 配信のタイムキーパーに

1. 配信終了予定時刻（例：22時00分）を設定
2. メッセージに「配信終了時間です」と入力
3. 残り時間がカウントダウン表示されます

### 作業リマインダーに

1. プリセットから「1時間ごと」を選択
2. タイマー追加ボタンをクリック
3. 1時間ごとに「○時間経過しました」と通知されます

## よくあるトラブルと解決方法

### タイマーが起動しない

- **確認1**: 棒読みちゃんが起動しているか確認
- **確認2**: Pythonがインストールされているか確認（コマンドプロンプトで`python --version`を実行）
- **確認3**: 設定ファイル（config/sample_config.json）が存在するか確認

### 音声が出ない

- **確認1**: 棒読みちゃんの設定で音声が出るか確認
- **確認2**: 棒読みちゃんのポート設定（デフォルト：50080）を確認

### 「タグのタイマーは既に起動しています」と表示される

- すでに同じタイマーが起動しています
- タスクバーを確認して、すでに起動しているタイマーを閉じてください

## 詳しい機能

- **複数タイマー対応**: 同時に複数のタイマーを設定できます
- **プリセット機能**: よく使う設定をかんたんに呼び出せます
- **棒読みちゃん連携**: 時間になると自動で音声通知します
- **重複防止**: 同じタイマーが重複して起動しないよう制御します

## 設定ファイルのカスタマイズ（上級者向け）

「config」フォルダ内の設定ファイル（JSONファイル）を編集することで、見た目や動作をカスタマイズできます。

```json
{
  "tag": "streaming_timer",
  "window": {
    "title": "配信タイマー",
    "width": 400,
    "height": 200,
    "background_color": "#000000"
  },
  "display": {
    "font_family": "メイリオ",
    "current_time": {
      "format": "%H:%M:%S",
      "font_size": 48,
      "color": "#00FF00",
      "label": ""
    },
    "remaining_time": {
      "format": "残り {hours}:{minutes:02d}",
      "font_size": 64,
      "color": "#00FF00",
      "label": ""
    }
  }
}
```
