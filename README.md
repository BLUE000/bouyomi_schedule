# BouyomiSchedule C++

定期的なスケジュールに基づいて「棒読みちゃん」に音声通知を送る、軽量な Windows デスクトップアプリケーションです。Python 版 `bouyomi_schedule` の C++ 移植プロジェクトです。

## 🌟 特徴
- **スタンドアロン**: Python ランタイム不要。EXE 単体（と DLL）で動作。
- **高精度タイマー**: 15分/30分/60分の区切り時刻を正確に計算。
- **柔軟な通信**: HTTP および TCP バイナリプロトコルによる棒読みちゃん連携。
- **低リソース**: C++ (Qt6) によるネイティブ実装。

## 🛠️ ビルド方法
Windows 環境で以下の手順でビルドできます。

1. **事前準備**:
   - Qt 6.10.1 (MinGW 13.1.0)
   - CMake
   - Ninja (推奨)

2. **ビルドの実行**:
   ```batch
   build.bat
   ```
   ※ `bin/` フォルダに `BouyomiSchedule.exe` が生成されます。

## 🧪 テストと評価
ロジックの正しさを検証するためのユニットテストを備えています。
```batch
cd build
ctest --output-on-failure
```

## 📂 ディレクトリ構成
- `api/`: 棒読みちゃん通信プロトコル実装
- `core/`: タイマー計算・スケジュールロジック
- `ui/`: Qt Widgets によるユーザーインターフェース
- `doc/`: 要件・設計ドキュメント
- `tests/`: ユニットテストコード
