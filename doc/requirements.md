# 要件定義書: BouyomiSchedule (C++ Port)

## 1. 目的
既存の Python 版 `bouyomi_schedule` を C++ (Qt6) に移植し、Python ランタイムを必要としない軽量・高速なスタンドアロンアプリケーションを提供する。

## 2. 対象ユーザー
- 配信者、ゲームプレイヤー、または定期的な音声通知を必要とする Windows ユーザー。
- PC に Python 環境を構築したくない、または最小限の構成でツールを動かしたいユーザー。

## 3. 機能要件

### 3.1 タイマー機能
- **現在時刻の表示**: リアルタイムな現在時刻の表示。
- **カウントダウン表示**: 最も近いスケジュール時刻までの残り時間を表示。
- **スケジュール登録**:
    - プリセット選択（15分/30分/60分単位の次区切り時刻）。
    - 任意時刻の直接入力。
- **スケジュール管理**:
    - 登録済みタイマーの個別削除機能。
- **通知実行**: 指定時刻になった際、あらかじめ設定されたメッセージを「棒読みちゃん」へ送信する。

### 3.2 棒読みちゃん連携機能
- **外部連携**: ネットワーク（TCP/HTTP）を介して「棒読みちゃん」へ読み上げ依頼を送信。
- **リトライ/繰り返し**: 設定された回数・間隔での繰り返し通知。
- **パラメータ指定**: 速度、音程、音量、声質の指定（オプション）。

### 3.3 設定管理
- **JSON設定**: Python版と互換性のある JSON 形式での設定読み込み。
- **二重起動防止**: ロックファイルを使用した単一インスタンスの保証。

### 3.4 ユーザーインターフェース要件
- **レイアウト**: スケジュール項目ごとに以下の構成を提供する。
    - `[声質プルダウン] [数値入力(中央)] [メッセージ入力(右端)] [削除ボタン]`
- **声質選択プルダウン**:
    - 以下の12種類を選択可能とする。
        - 声質: `y)`, `b)`, `h)`, `d)`, `a)`, `r)`, `t)`, `g)`
        - 特殊効果: `やまびこ)`, `エコー)`
        - パラメータ指定: `速度(値)`, `音程(値)`
    - 幅は選択肢の最大文字数に合わせた固定幅とする。
- **数値入力(中央)**:
    - 4桁の数値を入力可能とする。
    - **動的制御**: プルダウンで「速度(値)」または「音程(値)」が選択されている時のみ有効化し、それ以外は無効（操作不能）とする。
- **メッセージ入力(右端)**:
    - 読み上げメッセージを入力する。
    - ウィンドウ幅に応じて可変（伸縮）する。
- **削除ボタン**:
    - スケジュール行の右端に配置。
    - クリックすることで、対象のスケジュールを即座に削除する。

## 4. 非機能要件
- **ポータビリティ**: インストーラーを必要とせず、EXE と必要な DLL のみで動作すること。
- **低負荷**: バックグラウンド常駐を想定し、CPU/メモリ消費を最小限に抑えること。
- **ユーザーインターフェース**: 直感的で分かりやすい GUI の提供。

## 5. 動作環境
- OS: Windows 10 / 11
- 依存ソフトウェア: 棒読みちゃん (別プロセス)
